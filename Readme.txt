Установка
~~~~~~~~~

  Создайте подпапку в каталоге ...\Far\Plugins, скопируйте туда
  содержимое текущего архива.


Использование
~~~~~~~~~~~~~

  Запустите плагин:  F11 -> "Визуальное переименование файлов"
                     и прочитайте справку (нажмите F1).
